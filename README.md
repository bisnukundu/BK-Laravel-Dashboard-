# Bisnu Dashboard 

### Requirement
- Laravel: **Laravel 9**
- PHP: **php 8**


### Packages
- laravel/breeze
- livewire/livewire


### To enable Livewire add the following in your layout file: 
```html
<html>
    <head>
    ...
        @livewireStyles
    </head>
    <body>
    ...
 
        @livewireScripts
    </body>
</html>
```
